üîß Customization Guide

Change Colors

Edit tailwind.config.js:

theme: {
  extend: {
    colors: {
      primary: '#1565C0',    // Change primary blue
      secondary: '#00897B',  // Change secondary teal
    }
  }
}

Update Copy/Text

Each component has text as JSX. Example in Hero.tsx:

<h1>
  Software que protege su facturaci√≥n...
</h1>

Replace Images

In each component, find <img src="..."> and replace the URL.

Recommended sources: Unsplash, iStock, Pexels (home care focused)

Add New Sections

1. Create new component in components/NewSection.tsx

2. Import in App.tsx

3. Add to JSX: <NewSection />

‚ùì FAQ

Can I use this without React?

Yes! Open preview.html to see a standalone version that works in any browser.

How do I add real image from Unsplash?

Use the Unsplash API endpoint: https://api.unsplash.com/search/photos?query=home-healthcare&client_id=YOUR_KEY

Can I modify the design?

Absolutely! All colors, fonts, spacing are customizable through Tailwind CSS configuration.

How do I integrate a contact form?

The form in FinalCTA.tsx has an empty <form> tag. Add your API endpoint in the form's onSubmit handler.

Is this suitable for production?

Yes! This is production-ready code. It follows best practices for React, TypeScript, and modern web development.
