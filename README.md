# Performance Audit for ERP Application

## Prerequisites
- Node.js (v16+ recommended)
- npm or yarn
- Playwright

## Setup
1. Clone the repository
2. Run `npm install` or `yarn install`
3. Install Playwright browsers: `npx playwright install`

## Running the Audit
```bash
npm run audit
```

## Output
Performance audit results will be generated in `PERFORMANCE_AUDIT_REPORT.md`

## Notes
- Adjust test credentials and selectors as needed
- Some performance metrics require specific application configuration
